# SQLite Movie Project

## 📌 Project Overview
This project is a simple Python + SQLite database project where I practiced creating a database, inserting data, and running SQL queries.

The goal was to get hands-on experience with how Python interacts with databases.

## 🛠️ Tools Used
- Python
- SQLite (sqlite3 module)
- Git & GitHub

## 📂 What This Project Does
- Creates a SQLite database
- Builds a movies table
- Inserts sample movie data
- Runs SQL queries to retrieve and analyze data

## 📊 What I Learned
- How to connect Python to SQLite
- How to create and manage tables
- How to insert and query data using SQL
- How databases work at a basic level

## 🚀 Future Improvements
- Add more complex queries (filters, sorting, joins)
- Build a small UI or menu system
- Expand dataset beyond movies
